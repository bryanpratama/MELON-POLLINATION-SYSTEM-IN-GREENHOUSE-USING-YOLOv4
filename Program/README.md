# Model YOLOv4

Berikut konfigurasi dan informasi terkait model yang dilatih.

## Detail Model
- **Arsitektur**: YOLOv4
- **Dataset**: [Dataset Melon](../dataset/README.md)
- **Hasil Training**:
  - Precision: 0.97
  - Recall: 0.94
  - mAP: 96.76%

## File Penting
- **`yolov4-obj.cfg`**: Konfigurasi model YOLOv4.
- **`yolov4-obj.weights`**: Model terlatih.
- **`obj.names`**: Label kategori.
